# /photos/hero

This is the single photo that appears on the landing page, next to the
"A quiet stretch of West Coast" heading.

To change it: delete whatever's in here and drop in a new image
(.jpg, .jpeg, .png, .webp, .gif). Push, and the next Netlify build picks it
up automatically -- no code to edit.

If more than one image ends up in this folder, the first one alphabetically
is used, so it's best to keep just one here at a time.

A portrait-ish photo (taller than wide) will look best -- this slot crops to
a 4:5 shape.
